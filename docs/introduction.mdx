## What is the Blog Publishing System?

The Automated Blog Publishing System is a powerful, serverless solution that allows you to publish blog posts by simply sending JSON data to an API endpoint. The system handles everything from markdown conversion to GitHub integration.

## How It Works

<Steps>
  <Step title="Send JSON Data">
    Send your blog post data in JSON format to the API endpoint
  </Step>
  <Step title="Automatic Conversion">
    The system converts your JSON to Markdown with proper frontmatter
  </Step>
  <Step title="GitHub Operations">
    Creates a new branch, commits the file, and pushes to GitHub
  </Step>
  <Step title="Pull Request">
    Automatically creates a PR for review with labels and reviewers
  </Step>
  <Step title="Content Live">
    Once merged, content-collections automatically regenerates and your blog is live
  </Step>
</Steps>

## Quick Example

```json
POST /api/publish-blog
{
  "blogs": [
    {
      "title": "My First Blog Post",
      "description": "An introduction to our blog",
      "author": "John Doe",
      "date": "2025-10-17",
      "content": "# Welcome\n\nThis is my first post..."
    }
  ]
}
```

<Check>Response includes PR URL, branch name, and slug</Check>
